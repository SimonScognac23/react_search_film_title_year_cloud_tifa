In Italiano
Il progetto si trova nella cartella: cloud/tifa

Ho creato un'applicazione React per cercare film usando l'API OMDb. L'app permette di cercare film tramite una barra di navigazione e visualizzare i risultati in una lista. Quando clicco su un film, viene mostrata una pagina dettagliata con tutte le informazioni recuperate dall'API. Ho implementato due funzioni asincrone: fetchMovies per ottenere la lista dei film basata su una query di ricerca, e fetchMovieById per recuperare i dettagli completi di un singolo film usando il suo ID IMDb. Ho usato useState per gestire quattro stati: l'array dei film, il conteggio totale, i messaggi di errore e il film selezionato. Con useEffect ho fatto una chiamata automatica all'avvio dell'app per mostrare i "Guardians of the Galaxy" come ricerca predefinita. Ho aggiunto una validazione per impedire ricerche troppo corte (meno di 3 caratteri) e gestito il rendering condizionale per mostrare la lista o i dettagli del film a seconda dello stato. L'interfaccia usa Bootstrap per uno styling pulito e responsive.

In English
The project is located in the folder: cloud/tifa

I created a React application to search for movies using the OMDb API. The app allows searching for movies through a navigation bar and displaying results in a list. When I click on a movie, a detailed page is shown with all the information fetched from the API. I implemented two asynchronous functions: fetchMovies to get the list of movies based on a search query, and fetchMovieById to retrieve complete details of a single movie using its IMDb ID. I used useState to manage four states: the movies array, total count, error messages, and the selected movie. With useEffect, I made an automatic call on app startup to show "Guardians of the Galaxy" as the default search. I added validation to prevent searches that are too short (less than 3 characters) and handled conditional rendering to show either the list or movie details depending on the state. The interface uses Bootstrap for clean and responsive styling.
